# Sistema-de-Investimento
Tabela de Controle De Investimento. Identifica o FII( Fundo de Investimento) e a taxa correspondente ao FII, auxiliando o cotista a escolher a melhor opção que corresponde ao seu perfil de investidor. Dados baseados no site https://investidor10.com.br/fiis/ , com os ensinamentos do professor Felipe Aguiar.
